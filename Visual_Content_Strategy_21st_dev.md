# Visual Content Strategy with 21st.dev for API Monitoring SaaS

## Brand Visual Identity

### Color Palette
**Primary Colors:**
- **Trust Blue:** #2563EB (Primary brand color)
- **Uptime Green:** #10B981 (Success/positive indicators)
- **Alert Orange:** #F59E0B (Warning/attention needed)
- **Error Red:** #DC2626 (Critical alerts)
- **Neutral Gray:** #6B7280 (Text/background)

**Gradient Combinations:**
- Blue → Green (Trust → Success)
- Orange → Red (Warning → Critical)
- Blue gradient for hero graphics

### Typography
**Headlines:** Inter Bold (clean, modern, developer-friendly)
**Body Text:** Inter Regular (readable at small sizes)
**Code:** Fira Code (monospace for code snippets)

### Logo Usage
- **Full Logo:** Header graphics, hero images
- **Icon Only:** Social profile pictures, small graphics
- **Wordmark:** Where space allows

## Platform-Specific Specifications

### Twitter/X
**Image Sizes:**
- Single image: 1200×675px (16:9)
- Carousel: 1200×675px each
- Profile: 400×400px
- Header: 1500×500px

**Best Practices:**
- Text overlay readable at small sizes
- High contrast for mobile viewing
- Include brand colors in all graphics

### LinkedIn
**Image Sizes:**
- Post image: 1200×627px
- Article cover: 1200×644px
- Company logo: 300×300px
- Cover image: 1128×191px

**Best Practices:**
- Professional, clean design
- Include data visualization
- Use LinkedIn blue (#0077B5) as accent

### Instagram
**Image Sizes:**
- Square post: 1080×1080px
- Portrait: 1080×1350px
- Landscape: 1080×566px
- Stories: 1080×1920px

**Best Practices:**
- Visually striking
- Minimal text (Instagram prefers images over text)
- Use Instagram's brand colors as accents

### YouTube
**Thumbnail Size:** 1280×720px (16:9)
**Best Practices:**
- Bold, readable text
- Faces/emotion in thumbnails
- Consistent thumbnail style across series

### TikTok
**Video Size:** 1080×1920px (9:16)
**Best Practices:**
- Vertical format
- Captions/text overlay
- Trend-aware content

## Content Creation Workflow with 21st.dev

### Step 1: Content Planning
1. Identify content type (educational, promotional, etc.)
2. Choose appropriate template
3. Gather data/copy/images

### Step 2: 21st.dev Creation
1. Select template from library
2. Customize with brand colors
3. Add content/text
4. Adjust layout as needed

### Step 3: Quality Check
1. Verify brand consistency
2. Check readability at different sizes
3. Test on multiple platform previews
4. Ensure accessibility (color contrast, alt text)

### Step 4: Export & Optimization
1. Export in appropriate formats (PNG for graphics, MP4 for videos)
2. Optimize file sizes
3. Add platform-specific metadata
4. Schedule for posting
